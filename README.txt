This is an unofficial API for nairaland.com as one doesnt exist for now. I hope at least someone finds it useful in anyway possible. 
Consider using this API at your own risk and to your own detriment as its is an EXTREME WORK IN PROGRESS.

To run this program, you will need to have Python installed on your system: get that from www.python.org and also the 
AWESOME Requests: HTTP for Humans Python library. Get it from http://docs.python-requests.org/en/latest/index.html

I believe this program is simple enough that it wouldnt require a Getting Started or How it Works guide. #knockyourselfout.
